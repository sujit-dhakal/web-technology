5. **[HTTP Protocol](chapter-1/05-http-protocol.md)**
   HTTP requests, responses, methods (GET, POST, etc.), and status codes (200, 404, 500, etc.).

6. **[URL (Uniform Resource Locator)](chapter-1/06-url.md)**
   Anatomy of URLs including scheme, domain, port, path, query parameters, and fragments.

7. **[Client-Side and Server-Side Scripting](chapter-1/07-scripting.md)**
   JavaScript running in browsers vs PHP/Python running on servers, with code examples for both.

8. **[Web 1.0 and Web 2.0](chapter-1/08-web1-web2.md)**
   Evolution from read-only web (1991-2004) to interactive, social web (2004-present).
